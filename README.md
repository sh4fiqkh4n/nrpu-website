# NRPU 2026/27 Website

Static multi-page website for the National Research Program for Universities call information flow.

## Deploy on GitHub Pages

1. Upload all files in this folder to the repository root.
2. Go to Settings > Pages.
3. Select Deploy from a branch.
4. Choose `main` and `/root`.
5. Save and wait for the Pages URL.

## Main pages

- `index.html` — landing page and current call tracker
- `about.html` — program overview
- `eligibility.html` — PI, host institution and project readiness
- `apply.html` — application workflow and draft form
- `calendar.html` — horizontal timeline
- `resources.html` — official links and downloads
- `contact.html` — HEC/NRPU contact details
- `platform.html` — implementation model

## Notes

Replace `assets/hec-logo.svg` with the approved official artwork before formal institutional publication.
