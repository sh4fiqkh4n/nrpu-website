Platform

Content structure and production-ready patterns for a maintainable NRPU website.

Content model

CallYear, status, opening date, closing date, portal URL.
MilestoneDate, label, applicant action, status.
DocumentCategory, title, official URL, version, upload date.
FAQQuestion, answer, audience, last reviewed date.

Page behavior

  • Responsive navigation and mobile-first layout.
  • Current-date timeline calculation.
  • Filterable document library and searchable resources.
  • Semantic page titles, descriptions and structured data.
  • Accessible labels, skip link and keyboard-readable controls.

Implementation route

Static launch

Deploy HTML, CSS and JavaScript through GitHub Pages for fast review and stakeholder feedback.

CMS migration

Convert repeated sections into editable blocks: call banner, milestones, cards, downloads and FAQs.

Operations

Maintain a versioned document library, audit links, update call dates and archive previous cycles.
